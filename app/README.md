# Android SMS Auto Forwarding Service

An Android application that automatically forwards incoming SMS messages
based on keywords and sender phone numbers, with blacklist support.

---

## Features
- Background SMS forwarding service
- Keyword-based message filtering
- Sender phone number filtering
- Blacklisted phone numbers
- Blacklisted keywords
- Optimized for low battery usage
- Legacy Android support
- Custom UI layouts

---

## Use Case
Built to solve a real client requirement where specific types of incoming
messages need to be automatically forwarded to customers without manual action.
